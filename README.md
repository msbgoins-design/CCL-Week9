## Volunteer Dashboard 🌍

A dynamic, front-end React application designed to help users discover, filter, and save community volunteer opportunities. 

## ✨ Features
* **Dual Data Handling:** Seamlessly integrates external API data with a custom, locally-seeded mock database.
* **Advanced Filtering:** Users can filter opportunities instantly by search terms, categories, and specific locations.
* **Pagination Logic:** Implements "Load More" functionality to keep the UI clean while handling large datasets.
* **Local Storage Persistence:** Users can create custom volunteer opportunities and save their favorites directly to their browser's memory, ensuring data persists across page refreshes.
* **Modern UI:** Built with Bootstrap and custom CSS for a responsive, interactive, and highly stylized user experience.

## 🛠️ Tech Stack
* React.js (Vite)
* JavaScript (ES6+)
* HTML5 & CSS3
* Bootstrap

## 🚀 How to Run Locally
1. Clone the repository to your local machine.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.
